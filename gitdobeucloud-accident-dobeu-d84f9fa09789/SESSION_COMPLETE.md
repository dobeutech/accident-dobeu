## üîß Quick Start Guide

### For New Developers
```bash
# 1. Clone repository
git clone https://github.com/dobeutech/accident-dobeu.git
cd accident-dobeu

# 2. Setup backend
cd backend
npm install
cp .env.template .env
# Edit .env with your configuration

# 3. Validate environment
cd ..
./tools/env-check.sh

# 4. Run migrations
cd backend
npm run migrate

# 5. Start development
npm run dev

# 6. Access API docs
open http://localhost:3000/api-docs
```

### For Operations
```bash
# Production deployment
cd backend
npm ci --production
npm run migrate
pm2 start ecosystem.config.js --env production

# Verify deployment
curl http://localhost:3000/health/detailed
./tools/env-check.sh
./tools/deps-audit.sh
```
